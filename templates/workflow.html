Step-by-Step Logic Flow

1. Extract IOCs from APTnotes PDFs

python extract_pdfs.py

Parses PDFs into extracted IOCs → Data/extracted_pdfs/extracted_iocs.csv

2. Map IOCs to MITRE ATT&CK Groups

python map_iocs_to_attack.py

Maps extracted IOCs to threat groups → Data/mapped/group_ttps_detail.csv

3. Match Specific TTPs

python matching_test.py

Enter up to 5 TTPs to match → matched_groups.csv

4. Generate Intelligence Report

python report_generator.py

Generates professional DOCX report → Generated_Reports/Threat_Report_*.docx

Web App Usage

  1. Run: python app.py
  2. Visit: http://127.0.0.1:5000
  3. Input up to 5 TTPs → View Results → Export via CLI
